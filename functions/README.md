# Functions

This section will contain C programming exercises focusing on functions, including function definitions, calls, parameter passing, and return values.

## Questions Covered

1. To print SI and amount using functions
2. To print sum and average of 3 numbers using function.
3. To convert celsius into fahrenheit and vice versa using function
4. To convert seconds into hours, minutes and second.
5. To find sum of greatest and smallest element in an array using function.
6. To swap the values of two variables using function.
7. Calculate sum of two vectors using functions
        a) Using pass by value
        b) Using pass by reference
8. Insert an element into the desired location of a given array.
        a) Using pass by value
        b) Using pass by reference
9. Search for an element in a given array.
10. Perform matrix multiplication by passing it to array.
