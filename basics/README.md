# Basics

This section contains fundamental C programming exercises covering variables, operators, input/output, and basic calculations.

## Questions Covered

1. Program to print area and perimeter of rectangle
2. Program to calculate simple interest and amount
3. Program to print the volume of room and its surface area
4. Program to print the area of 4 walls and cost of painting at rs 270 per meter square
5. Program to print the area and perimeter of circle having any radius
6. Program to print sum, difference, product and average of any two numbers
7. Program to print cube of given number
8. Program to find out the square root of any number
9. Program to print sum of cubes of two numbers
10. Program to print normal, integer and modular division of any two numbers
11. Program to convert Celsius into Fahrenheit
12. Program to convert Fahrenheit into Celsius
13. Program to convert km into meter
14. Program to convert Nepali currency into USD
15. Program to convert seconds into hours, minutes and seconds
16. Program to convert Km into meter (duplicate of #13)
17. Program to convert inches into feet and inches
18. Program to convert days into years, months, weeks and days
19. Program to add last digit and second last digit of a given number
20. Program to calculate distance (s=ut+1/2at^2)
21. Program to find profit from given SP and CP
22. Program to input perpendicular and base of right angled triangle and compute hypotenuse
23. Program to find final velocity
24. Program to find area of triangle given its three sides
25. Program to solve quadratic equation given a, b and c
