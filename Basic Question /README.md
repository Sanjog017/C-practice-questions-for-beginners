# C Practice Questions for Beginners

This repository contains a collection of C programming exercises that I practiced while learning C. These questions cover fundamental concepts and help beginners get comfortable with basic programming constructs in C.

## About

These practice questions helped me develop a solid foundation in C programming. They cover topics such as:
- Basic input/output operations
- Arithmetic calculations
- Conditional statements
- Mathematical formulas
- Unit conversions
- Simple algorithms

## How to Use

1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/C-practice-questions-for-beginners.git
   ```

2. Navigate to the directory containing the questions

3. Compile any C program using gcc:
   ```bash
   gcc -o program_name program_name.c
   ```

4. Run the compiled program:
   ```bash
   ./program_name
   ```
